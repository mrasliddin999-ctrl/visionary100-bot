## Bot buyruqlari
- /start — Botni ishga tushirish + Telegram ID
- /maqsadlar — Barcha maqsadlar
- /bugun — Bugungi eslatma
- /statistika — Progress ko'rish
- /vaqt — Eslatma vaqtini o'zgartirish
- /stop — Eslatmalarni to'xtatish
